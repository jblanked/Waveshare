# Waveshare
This repository contains drivers and examples for various Waveshare devices. Waveshare is known for producing a wide range of hardware, including displays, sensors, and development boards.

## Currently Supported Devices:
- RP2350 1.28inch Round Touch LCD
- RP2350 1.43inch AMOLED Display Development Board
- RP2350 3.49inch Touch LCD
